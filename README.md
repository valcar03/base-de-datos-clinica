# base-de-datos-clinica
Una aplicación de escritorio para gestionar pacientes, citas y tratamientos en una clínica de podología.
  Características

- **Gestión de Pacientes**: Registro completo con información de contacto
- **Sistema de Citas**: Agenda y seguimiento de tratamientos
- **Archivo de Fotos**: Almacenamiento de imágenes de tratamientos
- **Etiquetas Inteligentes**: Sistema de categorización reutilizable
- **Asistente IA**: Búsquedas inteligentes en la base de datos
- **Interfaz Intuitiva**: Fácil de usar para profesionales de la salud

 Tecnologías

- Python 3.8+
- Tkinter (Interfaz gráfica)
- SQLite (Base de datos)
- Pillow (Procesamiento de imágenes)
